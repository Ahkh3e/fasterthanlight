Put your audio files in this folder structure:

- public/audio/music/menu.mp3
- public/audio/music/gameplay.mp3
- public/audio/music/battle.mp3

- public/audio/sfx/ui_click.mp3
- public/audio/sfx/lobby_join.mp3
- public/audio/sfx/laser_shot.mp3
- public/audio/sfx/explosion.mp3
- public/audio/sfx/planet_capture.mp3
- public/audio/sfx/faction_eliminated.mp3
- public/audio/sfx/game_over.mp3

Notes:
- Keep files short and compressed (mp3/ogg).
- If a file is missing, the game still runs; that sound just won't play.
- Paths are served by Vite as /audio/...
